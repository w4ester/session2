What are tokens?

LLMs process text as tokens (subwords/characters). Counting tokens helps you estimate prompt size and model limits. Chat-style prompts often add special tokens that represent roles and message boundaries.

What do the #numbers mean?

Each token has a unique ID number from a vocabulary of ~100,000 tokens. The model was trained to recognize these specific patterns:

  • Common words get single tokens (e.g., "the" = #1820, "marketing" = #90099)
  • Rare words are split into pieces (e.g., "tokenization" ‚Üí "token" + "ization")
  • Spaces/punctuation are often part of tokens (e.g., " hello" with a leading space)
  • Numbers & code follow special patterns (e.g., "123" might be 1 token or 3)
  • Emoji typically use multiple tokens per character

The same token ID always represents the same text, making it easy for the model to "understand" patterns across different prompts.

2) Prompt Engineering Basics

Four principles: be specific, provide context, use examples, and break complex tasks into steps.

Key Principles

  1. Be specific and clear ‚Äî specify format, tone, length, and audience.
  2. Provide context ‚Äî share relevant background and constraints.
  3. Use examples ‚Äî show a few samples of good outputs.
  4. Break tasks into steps ‚Äî guide the model‚Äôs reasoning.
