üìù Live Build Guide

Step-by-step written instructions - follow along or reference later

Step 1: Open GPT Editor (2 minutes)

  1. Go to chatgpt.com/gpts/editor
  2. You should see a split screen: configuration on left, preview chat on right
  3. If you see "Upgrade to Plus" - you need ChatGPT Plus to continue

Step 2: Name & Description (3 minutes)

Type this with me:

Name: Executive Communication Assistant

Description: Drafts board presentations, investor updates, and strategic briefs in [Your Company]'s voice. Uses data-driven insights and maintains a confident, clear executive tone.

Replace [Your Company] with your actual company name.

Step 3: Instructions (10 minutes)

Copy this template and customize:

You are an Executive Communication Assistant for [Your Company].

**Your Role:**
- Draft board presentations, investor updates, and strategic briefs
- Always use a confident, clear, executive tone
- Include data-driven insights and metrics
- Structure content for busy executives (top-line summary first)
- Ask clarifying questions before drafting

**Tone & Style:**
- Professional but not stuffy
- Data-driven and factual
- Strategic focus (big picture + key details)
- Concise - every word counts
- Action-oriented

**Content Structure:**
1. Executive Summary (3-5 bullet points)
2. Situation/Context
3. Data & Insights
4. Recommendations
5. Next Steps

**Before drafting, always ask:**
- What is the audience? (Board, investors, internal team?)
- What is the key message?
- What decisions need to be made?
- What data should be highlighted?

**Never:**
- Use jargon without explanation
- Make recommendations without data
- Write long paragraphs (use bullets)
- Assume context - ask questions

Customization Tips:

  • Replace [Your Company] with your company name
  • Adjust tone to match your style (more/less formal)
  • Add specific topics you frequently write about
  • Include any company-specific terminology or frameworks

Step 4: Upload Knowledge (8 minutes)

What to upload:

  • Past successful documents: 2-3 board decks or investor updates that represent your best work
  • Brand guidelines: Tone, voice, visual identity docs
  • Company data: Key metrics, mission/vision statements
  • Templates: Any standard formats you use

How to upload:

  1. Click "Knowledge" section in GPT Editor
  2. Click "Upload files"
  3. Select your documents (PDF, DOCX, TXT supported)
  4. Wait for upload to complete (shows green checkmark)

Don't have docs ready? Use our sample brand guidelines template to get started.

Step 5: Enable Capabilities (5 minutes)

Recommended settings:

  • ‚úÖ Web Browsing: Enable if you want GPT to research current data/trends
  • ‚úÖ DALL-E Image Generation: Enable if you want it to create simple diagrams
  • ‚ùå Code Interpreter: Usually not needed for executive comms

When to use each:

  • Web Browsing: "Research latest industry trends and include in this brief"
  • DALL-E: "Create a simple diagram showing our 3-year growth strategy"
  • Code Interpreter: "Analyze this CSV of sales data"

Step 6: Test Together (5 minutes)

Try these test prompts:

Test 1 - Basic Draft:

"Draft a 1-page investor update covering our Q4 performance:
- Revenue up 25% to $5M
- New enterprise customers: 12
- Product launch successful
- Raised $10M Series A"

Test 2 - Board Presentation:

"Create a 5-slide board presentation outline on our 2025 strategic priorities: expansion to Europe, new product line, and team growth."

Test 3 - With Questions (shows GPT asking for clarity):

"I need to write an update for our investors about a strategic pivot."

The GPT should ask clarifying questions before drafting.

Step 7: Save & Share (2 minutes)

  1. Click "Save" in the top right
  2. Choose visibility:
    • "Only me" - Private to your account
    • "Only people with a link" - Shareable link (recommended for team)
    • "Public" - Listed in GPT Store (not recommended for executive GPT)
  3. Click "Create" or "Update"
  4. Copy the GPT link to share with your team

üîß Troubleshooting Guide

Common issues during live builds and how to fix them

‚ùå "You need ChatGPT Plus to create GPTs"

Fix: Upgrade at chat.openai.com/plus ($20/month)

Backup: Take notes during the workshop and build later after upgrading

‚ùå Can't see GPT Editor interface

Fix:

  • Make sure you're logged into ChatGPT
  • Try a different browser (Chrome, Safari, Firefox)
  • Clear cache and cookies
  • Disable browser extensions that might interfere

‚ùå File upload failing

Fix:

  • Check file size (max 512MB per file)
  • Supported formats: PDF, DOCX, TXT, CSV, JSON, XML
  • Try converting to PDF if using other format
  • Upload one file at a time if multiple uploads fail

‚ùå GPT not following instructions

Fix:

  • Make instructions more specific and directive
  • Use examples in your instructions
  • Add "Never" or "Always" statements for critical behaviors
  • Test with different prompts to identify the issue

‚ùå GPT responses are too generic

Fix:

  • Upload more knowledge files (past work examples)
  • Add more specific instructions about your company/industry
  • Include tone/style examples in instructions
  • Provide more context in your prompts

‚ùå Can't share GPT with team

Fix:

  • Make sure you saved the GPT first
  • Choose "Only people with a link" visibility
  • Copy the full GPT URL (looks like: chatgpt.com/g/g-xxxxx)
  • Team members need ChatGPT Plus to use your GPT

‚ùå Lost connection / page crashed

Fix:

  • GPT editor auto-saves - refresh the page
  • Check your internet connection
  • If lost, use the written guide below to catch up

üí° Pro Tips for Success:

  • Don't try to make it perfect on first try - iterate!
  • Test thoroughly before sharing with team
  • Start simple, add complexity later
  • Keep a copy of your instructions in a doc (for backup)
